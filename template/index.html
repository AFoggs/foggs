Web Application

Academic

During the summer of 2018, I took a class focused on Data APIs. Specifically, we were working with public api data-sets and NodeJS in order to create web applications.

This being my first true time to make a web application, I decided to work with something that would be relevant: Weather data. I wanted to make an application that I could use even after the class was over.

I made use of an api key I had for weather underground. Working with NodeJS, pug, and elements of design, I created a mobile first weather application which pulls relevant data for certain cities in illinois.

This was my first project of this type. It could still use some work, but for now it is functional and serves as a good base. I have deployed the app through Heroku and it is publically facing for anyone to see.
