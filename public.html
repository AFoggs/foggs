Weather Web Application

Summer Academic Project

During the summer of 2018, I took a class focused on Data APIs. Specifically, we were working with public api data-sets and NodeJS in order to create web applications.

This being my first true time to make a web application, I decided to work with something that would be relevant: Weather data. I wanted to make an application that I could use even after the class was over.

I made use of an api key I had for weather underground. Working with NodeJS, pug, and elements of design, I created a mobile first weather application which pulls relevant data for certain cities in illinois.

This was my first project of this type. It could still use some work, but for now it is functional and serves as a good base. I have deployed the app through Heroku and it is publically facing for anyone to see and utilize.

Transgressions of the Heart

Poetry Book

We live. We love. We hurt. Transgressions of the Heart is a collection of poetry that explores the emotional of the heart. Divided into three parts, the passion, the falling, and the impact, Transgressions of the Heart explores the many stages of love. It is a story about meeting someone, falling in love, and then falling even further out of it.

I didn't realize it yet
but I got a lot more
than I ever intended to take

being naive and
not knowing that

the passion is always going to be the best part.

Transgressions of the Heart is my first poetry book that I published. It is a story of about life, love and loss. It was a year long journey, but I am satisfied with the outcome. It contains a poem for every state your heart may be in.

Raspbery Pi Project

Tech Project

After purchasing a Raspbery Pi, I decided that I wanted to work on some projects that improved the quality of my internet use at home. There was no need to do anything major, rather I focused on fixing a few minor pain points in my life.

I focused on what was important to me: improving security, upgrading my home network speed, and increasing accessibility. With these things in mind, I set out to complete this: Ad-blocking at the network level, Home VPN access, and Remote Access to my Raspberry Pi, with both GUI and Non Gui Options. Below are the tools I utilized:

  • Ad-Blocking via Pi.hole
  • VPN via PiVPN
  • Remote Linux Workstation W/ Gui via Real VNC Viewer
  • Remote Terminal Access via PuTTy

Now, my Raspberry Pi is a very useful item in my home. With the ability to block ads even on my mobile network no matter where I go, I am enjoying the end-result of my project. To learn how to create your own home pi performance server, check out my Pi Project on github.
